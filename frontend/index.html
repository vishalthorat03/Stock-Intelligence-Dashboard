About NSE Stock Intelligence

This project combines a Go API, Python scoring logic, SQLite storage, multi-source quote collection, and a lightweight dashboard to surface live market signals and explainable rankings.

Features

  • Whole-market style browsing from the stored stock universe
  • Search, pagination, ranking, and comparison charts
  • Market-wide retraining on every refresh
  • Explainable ranking and optional GPT-enhanced reasoning
  • Saved SQLite snapshots for every refresh cycle
